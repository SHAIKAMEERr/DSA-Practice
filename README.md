## 📖 Problem-Solving Approach
1️⃣ **Understand the problem statement** carefully.  
2️⃣ **Identify constraints and edge cases** to ensure robustness.  
3️⃣ **Plan the approach** (Brute Force ➡ Optimized Solution).  
4️⃣ **Write clean and efficient code** following best practices.  
5️⃣ **Analyze time and space complexity** to evaluate performance.  
6️⃣ **Refactor and optimize** for better efficiency.  
7️⃣ **Test thoroughly** with different edge cases before submission.
